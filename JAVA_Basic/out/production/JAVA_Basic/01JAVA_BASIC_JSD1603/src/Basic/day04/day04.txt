笔记:
1.循环:反复执行一段相同或相似的代码
2.循环三要素:
  1)循环变量的初始化
  2)循环的条件(以循环变量为基础)
  3)循环变量的改变(向着循环的结束变)
  循环变量:在循环过程中所改变的那个量
3.循环结构:
  1)while:先判断后执行，有可能一次都不执行
  2)do...while:先执行后判断，至少执行一次
        第1要素与第3要素相同时首选
  3)for:应用率最高的
4.break:跳出循环
  continue:跳过循环体中剩余语句而进入下一次循环

1.顺序结构:从上往下逐行执行，每句必走
